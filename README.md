<!-- Getting Started -->
## 	:toolbox: Getting Started

<!-- Prerequisites -->

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/jasontranpk/project-reactjs-places
```

Go to the project directory

```bash
  cd project-reactjs-places
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
